1、Set集合

当传入相同的值作为参数，那么后续的调用实际上会被忽略

```javascript
let result=[2, 3, 5, 9, 4, 5, 7, 10]; //result.length=8
console.log(new Set(result)) // Set(7) {2, 3, 5, 9, 4, 7, 10}
```

2、Array.from()

将对象转换为数组

```javascript
console.log(Array.from(new Set(result))) //[2, 3, 5, 9, 4, 7, 10]
```

所以Set集合与Array.from()连用就可以帮助数组去重
