```javascript
# 2

```

方法2：**Math.floor()**

返回小于等于x的最大整数

```javascript
# 1
Math.floor(1.6);
```

2、连接两个数组

方法1：**concat函数**，不会改变现有的数组

```javascript
let a = [1, 2, 3];
let b = [0];

# [1,2,3,0]
console.log(a.concat(b))
```

方法2：**[...arr1,...arr2]**

```javascript
let a = [1, 6];
let b = [0];
let numList = [...a, ...b]
console.log(numList)
```
