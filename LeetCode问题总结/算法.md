## 二分法

`left`必须是`mid+1`

```javascript
const binarySearch = (list, target) => {
  let left = 0,
    right = list.length - 1;
  while (left < right) {
    const mid = left + Math.floor((right - left) / 2);
    if (list[mid] > target) {
      right = mid;
    } else {
      left = mid + 1;
    }
  }
  return list[left];
};
```
